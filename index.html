SUCCSS
Succss! Easy to install, easy to use CSS regression testing tool.
 Turn your Cascading StlyeSheets into successful design integrations.
Tweet
Grab it: 
npm install succss
Succss transparently use PhantomJS, Casperjs and Imagediff npm dependencies to work. Follow this guide if you have trouble with npm.
Describe your website CSS: 
// data.js
  Succss.data = {
    'SUCCSS selftests': {
      url:'succss.ifzenelse.net',
      baseDirectory:'screenshots',
      'home': {
        // Key:name Value:CSS selector.
        captures: {
          'header':'header',
          'square1':'#red.color-square',
          'square2':'#green.color-square',
          'square3':'#blue.color-square'
        },
      },
      viewports: {
        'classic wide': {
          width: 1366,
          height: 768
        },
        'mobile': {
          width: 320,
          height: 480
        }
        ...
  }
Succss can be used in many ways, for many use cases. Later, you can also read Advanced Configuration.
Take base screenshots: 
succss add data.js
The command line tool can additionally take several arguments, such as filters and options.
Read it all here !
Compare base screenshots to updates: 
succss check data.js
You can test your design with filters and options as well.
See differences or success: 
img img img
